## Install

* [Download](https://github.com/Holastor/Blade-and-Soul-2-Localization/releases/tag/4.0.0) Latest Release;
1) Скачайте Russian localization.zip;

2) Распакуйте в .\AION2_TW\Aion2\Content;

3) That's all folks!

## OR Source-Install
 1) Скачайте Source code (zip) [Download](https://github.com/Holastor/AION-2-Localization/archive/refs/heads/main.zip)

 2) В архиве найдите папку L10N;

 3) Скопируйте папку L10N в папку .\AION2_TW\Aion2\Content;

 4) Удалить папку .\AION2_TW\Aion2\Content\Paks\L10N\Text\en-US

 5) That's all folks!
